cmd命令:
1.在一个文件夹中shift+鼠标右键  在这个文件夹中打开cmd命令窗口
2.md first 创建一个first的文件夹
3.cd first 进入first文件夹中
4.echo>1.txt 创建1.txt文件
5.dir 查看当前目前的所有文件
6.del 1.txt 删除1.txt文件
7.md aaa 创建aaa文件夹
8.rd/s/q aaa 不用确认删除aaa文件夹  /s表示目录不是空的也可以删,/q表示安静地  类似linux的rm -rf
7.cd ..返回上一级目录

npm命令:
在first目录中npm init 创建一个项目(npm init --yes走默认流程)
1.会出现 name:(first) 回车即可 小写
2.version :(1.0.0) 输入0.0.1 表示第一个版本的
3.descripion(项目的描述) : Gulp course by ninghao.net
4.entry point : 回车
5.test command(测试命令): 回车
6.git repository(git的版本库) : 回车
7.keywords :回车
8.author :
9.license(许可的协议) :回车
10.然后会创建一个package.json文件
11.atom .  表示用atom打开当前的目录
12.npm install gulp --save-dev 会生成node_modules文件夹,里面有很多不需要的文件
并在在package.json中添加gulp的依赖包
13.npm install 下载package.json里面的包  放在新生成的node_modules文件夹中
14.ctrl+n 新建文件
15.var gulp = require('gulp') 得到gulp对象
16.gulp.task('hello',function(){ return ... })  注册gulp hello任务
17.gulp的拷贝命名会自动检查文件的重复性,如果destination目标文件夹中已经有了同名文件,那么会删除开始的文件
18.npm install gulp-sass --save-dev 安装sass插件到项目中

其他.npm uninstall babel-cli -g 从全局删除babel-cli的命令
