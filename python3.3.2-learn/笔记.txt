1.在vscode上安装python
2.新建一个py文件,ctrl+r,然后提示创建任务
3.或者直接拷贝任务：
在项目中新建.vscode目录,
新建settings.json可以快速打开vscode的工作区设置(本项目)覆盖用户区设置(其他项目默认的配置)

配置pylint:
pip install pylint 然后pylint就会按照到系统默认的python第三方库里面
pip install yapf
settings.json:
{
  "python.pythonPath": "C:\\Users\\Administrator\\AppData\\Local\\Programs\\Python\\Python36\\python.exe",
  "python.linting.pylintEnabled": true,
  "python.linting.pylintPath": "pylint",
  "python.formatting.provider": "yapf",
}
